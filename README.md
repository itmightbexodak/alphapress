# ambercial ALPHAPRESS
### Powered by FreeBSD

알파프레스(ALPHAPRESS)는 FreeBSD를 기반으로 둔 데스크탑/서버용 운영체제입니다.

더욱 안정적인 Unix 환경을 경험해보세요.

## 빌드 및 재배포 방법

ALPHAPRESS(이하 APOS)의 라이브 부트/설치 이미지는 공식 사이트나 깃허브 리포지토리 등에서 받을 수 있으나, 최신 FreeBSD 환경 내에서 단순한 스크립트 실행 한번만으로 나만의 APOS 환경을 구축할 수 있습니다.

1. 리포지토리 클론

``git clone https://github.com/itmightbexodak/alphapress``
리포지토리를 클론합니다.

그 후 ``cd alphapress`` 명령으로 APOS 리포지토리에 들어간 후 ``sh apbuild.sh``로 빌드를 시작합니다.

(팁!| 스크립트 내부에는 기본적으로 탑재할 패키지들을 명시해둔 부분이 있습니다. 그 부분을 수정하여 환경 커스터마이징이 가능합니다.)
2. 행운을 빌기

진짜로 따라하실 지는 모르겠지만 행운을 빕니다! :-)

3. 즐기기

스크립트 내 명시된 경로에 빌드된 ISO 파일이 생겼을 겁니다. 이제 즐기세요!
